# electron-music
基于react16 electron封装的音乐播放器

## 功能点

* 播放
* 播放歌曲页面
* 登录注册
* 个人歌单（歌曲）
* 推荐
* 榜单
* 歌手
* MV
* 播放列表
* 软件基本功能
* 搜索
* 分享
* 音乐库
